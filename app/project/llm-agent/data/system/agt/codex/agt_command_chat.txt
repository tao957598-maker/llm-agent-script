# AC 客户端操作指令

## 命令说明表格

| 命令 | 行为 | 返回 JSON 格式 |
|------|------|----------------|
| `PowershellInterrupt` | 中断当前的 PowerShell 终端会话 | `{"command": "PowershellInterrupt"}` |
| `PowershellView` | 查看当前 PowerShell 终端会话的内容或状态 | `{"command": "PowershellView"}` |
| `PowershellClose` | 关闭当前的 PowerShell 终端会话 | `{"command": "PowershellClose"}` |
| `PowershellCommand` | 在 PowerShell 终端中执行指定的命令 | `{"command": "PowershellCommand"}` |
| `SaveDesktopScreenshot` | 截取当前桌面屏幕截图并保存到文件系统 | `{"command": "SaveDesktopScreenshot"}` |

## 意图识别规则（按优先级从高到低）

1. **截图**：如果用户输入包含“截图”、“截屏”、“桌面截图”等 → `{"command": "SaveDesktopScreenshot"}`

2. **关闭终端**：如果用户输入包含“关闭终端”、“终止终端”、“结束终端”等 → `{"command": "PowershellClose"}`

3. **中断命令**：如果用户输入包含“中断”、“停止执行”、“Ctrl+C”等 → `{"command": "PowershellInterrupt"}`

4. **执行命令**：如果用户输入包含明确的**执行动作**（如“执行”、“运行”、“输入”）**并且**后面跟着具体的命令内容（即使内容看起来不是有效命令，也按执行处理） → `{"command": "PowershellCommand"}`

5. **查看终端**：如果用户输入包含“查看终端”、“终端内容”、“终端状态”、“终端输入”等**查看**意图 → `{"command": "PowershellView"}`

6. **默认兜底**：如果仅提到“终端”但没有明确“查看”、“执行”、“关闭”等上下文 → 默认返回 `{"command": "PowershellView"}`

| 用户输入 | 识别逻辑 | 返回命令 |
|----------|----------|----------|
| `终端` | 仅提到终端，无其他上下文（规则6） | `PowershellView` |
| `查看终端` | 明确查看意图（规则5） | `PowershellView` |
| `终端输入"终端"` | 包含“输入”+具体内容（规则4） | `PowershellCommand` |
| `执行 dir` | 包含“执行”+命令内容（规则4） | `PowershellCommand` |
| `运行 Get-Process` | 包含“运行”+命令内容（规则4） | `PowershellCommand` |
| `关闭终端` | 明确关闭意图（规则2） | `PowershellClose` |
| `中断当前任务` | 明确中断意图（规则3） | `PowershellInterrupt` |
| `截图` | 明确截图意图（规则1） | `SaveDesktopScreenshot` |
