# 函数命名规范
✅ **策略**：
- 函数名必须**语义清晰 + 唯一标识操作目标**。
- 避免仅用通用名如 `input()`、`click()`，而应结合**操作类型 + 元素语义**。
- 可通过以下方式保证唯一性：
  - 根据`current step`使用元素的业务语义（如 `username`, `password`, `login_button`）
  - 若同一元素有多种操作（如清空+输入），可加前缀/后缀（如 `clear_and_input_username`）

> 💡 **经验法则**：看到函数名就能知道“对哪个元素做了什么”。

---

###   函数名称应基于：“操作 + 元素语义”

✅ **推荐命名规范**：
