# json结构解释
## 顶层字段
*   **`columnInfos`**: 表内各列的定义信息数组，是核心数据结构。
*   **`comment`**: 表的注释说明，描述此表的用途,必须为英文。
*   **`displayOrder`**: 表的显示顺序，用于前端排序，数字越小通常排序越靠前。
*   **`tableName`**: 表的实际名称，必须以`kbs_`开头，后跟模块名、实体名、动作名。

---

## `columnInfos` 下的字段 (以示例的`id`列为例)
*   **`columnDefault`**: 该列的默认值。此处为空字符串，表示无默认值。
*   **`columnType`**: 英文小写,该列的数据类型只有: ["varchar","float","bigint","int","datetime","text","json"]；一般情况下优先使用`varchar`
*   **`displayOrder`**: 该列在前端显示的排列顺序，数字越小通常位置越靠前。
*   **`index`**: 是否为索引列。`false`表示此列未建立索引。
*   **`keyType`**: 键类型标识。仅允许`0`、`1`、`2`、`4`；其中`0`表示none，`1`表示主键（仅`id`字段可用），`2`表示uniqueIndex，`4`表示normalIndex；禁止使用`3`。由于最终会删除`id`字段，实际输出中仅应出现`0`、`2`、`4`。
*   **`maxLength`**: 该字段允许的最大长度（尤其针对字符串类型）。此处为`null`，表示对整型不适用。
*   **`name`**: 该列的字段名称，即表中的列名。
*   **`nullable`**: 该字段是否允许为空（NULL）。`false`表示不允许为空，是必填项。

n_result,act_func.run?script='''
def json = n_get_data
// 设置schema默认为llm_agent
json.schema = "llm_agent"

// 删除columnInfos中name为"id"的项
json.columnInfos = json.columnInfos.findAll { it.name != "id" }
// 检查是否包含created_at
def hasCreatedAt = json.columnInfos.any { it.name == "created_at" }
if (!hasCreatedAt) {
    json.columnInfos.add([
        name: "created_at",
        columnType: "datetime",
        columnDefault: "",
        comment: "Timestamp when this user record was created.",
        nullable: false,
        keyType: 0,
        index: false,
        maxLength: "",
        displayOrder: 0,
        vectorType: "",
        graph: false,
        vectorField: ""
    ])
}

// 检查是否包含updated_at
def hasUpdatedAt = json.columnInfos.any { it.name == "updated_at" }
if (!hasUpdatedAt) {
    json.columnInfos.add([
        name: "updated_at",
        columnType: "datetime",
        columnDefault: "",
        comment: "Timestamp when this user record was last updated.",
        nullable: false,
        keyType: 0,
        index: false,
        maxLength: "",
        displayOrder: 0,
        vectorType: "",
        graph: false,
        vectorField: ""
    ])
}

// 为每个字段添加缺失的属性
json.columnInfos.each { item ->
    item.vectorType = item.containsKey('vectorType') ? item.vectorType : ""
    item.graph = item.containsKey('graph') ? item.graph : false
    item.vectorField = item.containsKey('vectorField') ? item.vectorField : ""
    def normalizedColumnType = item.columnType != null ? item.columnType.toString().trim().toLowerCase() : "varchar"
    normalizedColumnType = [
        "string": "varchar",
        "varchar": "varchar",
        "float64": "float",
        "float": "float",
        "int64": "bigint",
        "uint64": "bigint",
        "bigint": "bigint",
        "integer": "int",
        "int": "int",
        "datetime": "datetime",
        "text": "text",
        "json": "json"
    ].get(normalizedColumnType, "varchar")
    item.columnType = normalizedColumnType
    item.keyType = item.keyType != null ? item.keyType : 0
    item.keyType = (item.keyType instanceof Number) ? item.keyType.intValue() : item.keyType
    item.keyType = item.keyType == 3 ? 4 : item.keyType
    item.keyType = (item.name != "id" && item.keyType == 1) ? 0 : item.keyType
    item.keyType = [0, 1, 2, 4].contains(item.keyType) ? item.keyType : 0
    item.maxLength = item.maxLength != null ? item.maxLength : ""
    item.comment = item.name
}

// 重新排序displayOrder
json.columnInfos.eachWithIndex { item, index -> item.displayOrder = index }

return json
'''
