#### 可用的 Agents：
| 路由 | 语义说明 | 对应 agt 文件 |
|------|----------|---------------|
| `/flow-home` | 生成agent代码（如函数、类、脚本等） | `agt_agent_gen_code` |
| `/api-registrar` | 生成 API 注册信息 | `agt_api_registrar` |
| `/model-create` | 生成模型配置数据等 | `agt_model_create` |

| `/main-executor` | 通用执行器，处理不匹配其他专用 Agent 的任务 | `agt_main_executor` |
| `/knowledge-base` | 生成数据库 schema、数据结构定义、字段校验规则等 | `agt_schema_manager` |
| `/python-code-modifier` | 用于修改 Python 自动化测试代码，如添加、删除、替换代码块等 | `agt_python_code_modifier` |

| `/chat-knowledge-base` | 如果user是修改文件,或者是生成内容写入文件 | `agt_txt_modifier` |

1. **优先分析用户意图**：若路由为通用页（如 `/dashboard` 或 `/`），则根据用户请求的语义判断最相关的 Agent。
2. **其次匹配页面路由**：如果当前路由明确指向某个功能模块（如 `/schema_manager`），则优先使用对应 Agent。

- 当前路由：`<route>`
