# Cursor CLI 命令行工具

Cursor 是基于 AI 的代码编辑器，支持通过命令行进行操作。

## Cursor 安装与启动

### macOS
```bash
# 使用 Homebrew 安装
brew install cursor

# 或下载 DMG 文件从 https://cursor.sh
```

### Windows
```bash
# 使用 winget 安装
winget install Cursor.Cursor

# 或下载 .exe 安装程序从 https://cursor.sh
```

### Linux
```bash
# 使用 npm 安装
npm install -g cursor-ai

# 或下载 AppImage
```

## Cursor CLI 基础命令

### 启动 Cursor
```bash
# 启动 Cursor（默认）
cursor .

# 启动并打开特定文件
cursor filename.txt

# 启动并打开多个文件
cursor file1.js file2.py

# 启动指定目录
cursor /path/to/project

# 使用特定窗口打开
cursor --new-window /path

# 等待窗口关闭后退出
cursor --wait
```

### 高级启动选项
```bash
# 以安全模式启动（禁用扩展）
cursor --safe-mode

# 以扩展开发模式启动
cursor --extension-development-host

# 禁用 GPU 加速
cursor --disable-gpu

# 使用远程服务器
cursor --remote ssh://user@host

# 指定用户数据目录
cursor --user-data-dir ~/.cursor-test
```

## Cursor AI 命令

### @cursor/sdk 集成
```bash
# 安装 Cursor SDK
npm install @cursor/sdk

# 或使用 Cursor CLI 初始化
cursor init
```

### AI 助手调用
```bash
# 触发 AI 补全
cursor "请解释这段代码"

# AI 代码审查
cursor --review "path/to/file"

# AI 重构建议
cursor --refactor "path/to/file"
```
