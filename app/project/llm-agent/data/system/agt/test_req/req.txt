# 公告管理

## 功能说明
管理系统中的公告，提供查询、新增、修改、查看、分析功能。

## 操作流程
1. 用户登陆系统后，通过菜单依次进入【系统管理】-【公告管理】功能界面。
2. 点击【新增】，弹窗展示新增公告页面，可选择定时发布、实时发布，定时发布需要设置发布时间，公告通知对象可选择机构或者员工，确定通知对象后，点击【选择】进行勾选公告通知的对象，公告信息填写完成后点击【确定】即可新增公告。
3. 点击【修改】，弹窗展示公告详细信息，调整公告信息后点击【确定】更新公告。
4. 点击【删除】，确认后删除选中的公告。
5. 点击【分析】，弹窗展示公告阅读人数的分析情况和列表。

## 业务规则
- 查询只能搜索本人创建的公告。
- 编辑公告信息时，必须满足当前时间 < 发布时间 < 到期时间，已发布的公告不可修改。
- 公告状态每天日终更新。
  - 发布时间 > 当前时间，公告状态更新为发布；
  - 到期时间 > 当前时间，公告状态更新为失效。

## 操作角色及权限

| 层级 | 角色名称 | 菜单权限 | 按钮权限 | 数据权限 |
|------|----------|----------|----------|----------|
| 总行 | 系统管理员 | √ | 全部 | 全部 |
| 总行 | 总行管理岗 | × | 无权限 | 无权限 |
| 分行 | 分行管理岗 | × | 无权限 | 无权限 |
| 支行 | 支行管理岗 | × | 无权限 | 无权限 |
| 支行 | 执行岗(客户经理) | × | 无权限 | 无权限 |

**数据权限**：全部、本级及下级（含指定机构及下级）、本级（含指定级）、本人。  
**按钮权限**：不做按钮权限。
