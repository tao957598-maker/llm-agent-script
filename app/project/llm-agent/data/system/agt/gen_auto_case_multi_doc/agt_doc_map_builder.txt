// Normalize document pages input
n_pages_normalized,act_func?script='''
    def raw = document_pages
    if (raw == null) raw = []
    if (!(raw instanceof List)) raw = [raw]
    def out = []
    raw.each { r ->
        if (!(r instanceof Map)) return
        def content = (r.get("content") ?: "").toString().trim()
        if (!content) return
        out.add([
            page: (r.get("page") ?: "").toString().trim(),
            content: content,
            source_ref: (r.get("source_ref") ?: r.get("source") ?: "").toString().trim()
        ])
    }
    if (out.isEmpty()) {
        println("[WARN][MAP·单文件] document_pages 为空（可能 .doc 解析失败），跳过该文档")
        fn.put("_doc_skip", "true")
    }
    return out
'''

'''

// Validate map result — 格式不对直接失败，不浪费时间重试
n_doc_map_validate,act_func?script='''
    def raw = n_doc_map
    def m = null
    if (raw instanceof Map) {
        def inner = raw.get("value")
        m = (inner instanceof Map && inner.get("modules") instanceof List) ? inner : raw
    } else if (raw instanceof List && raw.size() > 0 && raw[0] instanceof Map) {
        m = raw[0]
    }
    if (m == null || !(m instanceof Map)) {
        println("[ERROR][MAP·单文件] LLM 返回无效 map — 文档内容可能无法解析（乱码/格式异常），跳过 " + "{{documentName}}" + "，请检查源文档")
        return [platform: "{{platform}}", document_name: "{{documentName}}", modules: [], _meta: [document_name: "{{documentName}}", platform: "{{platform}}", module_count: 0, submodule_count: 0, input_chars: 0, skipped: true, skip_reason: "LLM returned invalid JSON — document content may be garbled or unparseable"]]
    }
    def modules = m.get("modules")
    if (!(modules instanceof List) || modules.isEmpty()) {
        println("[WARN][MAP·单文件] map.modules 为空（文档无有效模块），跳过 " + "{{documentName}}")
        m.put("_meta", [document_name: "{{documentName}}", platform: "{{platform}}", module_count: 0, submodule_count: 0, input_chars: n_input_guard, skipped: true, skip_reason: "No modules extracted from document"])
        return m
    }
    int totalSubmodules = 0
    modules.each { mod ->
        if (!(mod instanceof Map)) return
        def subs = mod.get("submodules")
        if (subs instanceof List) totalSubmodules += subs.size()
    }
    if (totalSubmodules == 0) {
        println("[WARN][MAP·单文件] 子模块总数为 0，跳过 " + "{{documentName}}")
        m.put("_meta", [document_name: "{{documentName}}", platform: "{{platform}}", module_count: modules.size(), submodule_count: 0, input_chars: n_input_guard, skipped: true, skip_reason: "Modules extracted but no submodules"])
        return m
    }
    m.put("_meta", [
        document_name: "{{documentName}}",
        platform: "{{platform}}",
        module_count: modules.size(),
        submodule_count: totalSubmodules,
        input_chars: n_input_guard
    ])
    return m
'''

# 约束与边界条件提取（重要）

从页面内容中识别以下类型的**非功能性但可测试的约束条件**，提取到对应子模块的 `constraints` 字段：

- **时间窗口/截止时间**：如 "before 4:55 PM"、"within 2 hours"、"same-day processing"
- **金额阈值**：如 "> HKD 5M"、"maximum 10 transactions"
- **数据保留期限**：如 "7 years"、"90 days"
- **SLA/性能承诺**：如 "24x7"、"response time < 2 seconds"
- **合规/安全要求**：如 "data encryption required"、"access control"、"audit trail mandatory"

每个约束条目包含：
- `type`：约束类型（从上述列表选择，不确定则用 "other"）
- `value`：约束的具体值（原文中的精确表述）
- `condition`：触发条件或适用场景
- `source_page`：原文所在页码

**重要**：不要编造不存在的约束。如果页面中没有明确的可测试约束，`constraints` 可以为空数组 `[]`。只提取原文中明确写出的数值、时间、期限、频率等可度量的约束。

# 必须遵守的硬约束

1. 只输出 JSON 本身。不要输出 ```json 代码块包裹、不要解释文字、不要 markdown。输出的第一个字符必须是 `{`，最后一个字符必须是 `}`。
2. `modules` 数组不能为空。如果文档确实没有可提取的模块，仍然返回空 `modules: []`，不要省略该字段。
3. 每个 submodule 必须至少有一个 `source_pages`，格式为 "PageN"（N 为数字）。
4. 所有用户可见文本（module_name、submodule_name、description、capabilities、business_rules、acceptance_criteria）必须使用 {{outputLanguage}}。constraints 的 value 字段应保留原文精确表述。
5. 保持描述简洁（每个条目 1-3 句话），不要复制大段原文。
6. 确保 JSON 语法完整——所有字符串用双引号，数组和对象括号必须配对闭合。
''')
