# 智能体代码文件说明

1.文件名必须以`agt`开头`.txt`结尾

2.文件内容结构:
开头`/* ... */`是定义智能体的参数和说明

文件末尾定义与GPT对话,必须是`define`接收两个参数:第一个是`tpl`开头的变量名,第二个参数是字符的提示词
`define('tpl_chat_bot', ''' xxxx ''')`

3.`{{变量名称}}`是变量占位符,在提示词中使用,会被替换为实际的值

4.其余的agent语法在`ai_agent_code/agent_grammar.txt`文件内

实例文件内容如下:
```
/*

			"name": "docContent",
			"label": "待修改的内容"
		},
		{
			"name": "kbSpaceFileId",
			"label": "文件ID",

		}
	],
	"version": 1
}
```
